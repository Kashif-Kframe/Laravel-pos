- Run below commands.

```sh
composer install
```
and

```sh
npm install
```


## Setup

- First of all please add the database credentails in.env file, (also set smtp settings for email sending)
- Then run the migration command to migrate all tables like

```sh
php artisan migrate
```
- After run the migration command, run the seeder command to seed the admin panel as well as company and employee data in tables e.g


```sh
php artisan db:seed
```
- Run below command to link storage for file storage.

```sh
php artisan storage:link
```

- To server then project please run the following command.

```sh
php artisan serve
```
- 
- Now you can log in as admin with following credentials.

```sh
Email: admin@gmail.com
Password: 12345678
```

- Now you can hit the url in browser to run the project.
